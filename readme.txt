Generate secure, branded PDF product catalogs from WooCommerce product data.

== Description ==

PDF Product Catalogs for WooCommerce adds a dedicated admin screen under Products where store operators can generate standard or client-specific PDF product catalogs.

Each catalog is generated from live WooCommerce product data and stored in a protected uploads directory. Historical files remain downloadable for authorized admin users from the catalog history table.

== Installation ==

1. Upload `pdf-product-catalogs-for-woocommerce.zip` in WordPress.
2. Activate the plugin.
3. Open `Products > PDF Product Catalogs`.
4. Configure the catalog defaults and generate your first PDF.
